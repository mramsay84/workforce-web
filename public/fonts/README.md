# Aileron Font Files

Place the self-hosted Aileron font files here:

- `Aileron-Regular.woff2` — weight 400
- `Aileron-SemiBold.woff2` — weight 600
- `Aileron-Bold.woff2` — weight 700

These files are loaded via `@font-face` in `src/app/globals.css`.
Aileron is available at https://dotcolon.net/font/aileron/

The site will fall back to system fonts if the Aileron files are absent.
